# 西中原中学校同窓会
2026-02-21 (土) 開催予定の還暦同窓会のためのページです。
学年全体の大きな同窓会は前回はゴールデンウィーク中の[2016-04-29 (金)(YouTube)](https://youtu.be/nKRteGkaig0) の開催でした。
大勢に参加いただけてとても楽しかったのですが、幹事側の準備は大変でした。
連絡先の調査から始まって、電話・メール・LINE・Facebookなど複数の連絡手段を使い分け、それでもリーチしない方への郵便での出欠確認などなどです。
今回は連絡手段をLINEに一本化することで簡略化したいと考えています。

  - 二次会のみ参加：5,000円
  - 両方参加：15,000円
